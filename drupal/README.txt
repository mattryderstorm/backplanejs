backplanejs for Drupal 6.x
==========================
This module is part of the backplanejs project, at:

  <http://backplanejs.googlecode.com/>


FEATURES
--------

* Allows one or more backplanejs libraries to be managed, along with multiple versions for each.
* Allows setting of backplanejs support for all nodes of a particular type, or indivual nodes.


BUG REPORTS
-----------

Post bug reports and feature requests to the issue tracking system at:

  <http://code.google.com/p/backplanejs/issues/list>

(Please use the tag 'Module-Drupal'.)
