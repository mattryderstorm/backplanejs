Description:
------------------------------------------------------------------------------------------------

Module allows storage of a LIST of RDFa datasource URLs in a DB table, 'rdfa_ds'.


Installation:
------------------------------------------------------------------------------------------------

Place the module files in either:

<drupal_home>/sites/all/modules to make available to all sites on server

OR 

<drupal_home>/sites/<site_name>/modules to make available to a specific site.

Enable/configure the module in the Drupal admin pages.
