This page simulates a blogger mentioning that they have an item for sale, and recommending two books. The blogger need only add a small amount of RDFa, and the Ubiquity library does the rest:

  • a widget is embedded that gives details of current bids for any items for sale;
  • a Yowl notification is shown for each eBay item found on the page;
  • a Yowl notification is shown for each book mentioned on the page;
  • a data request is sent to the eBay API for more information on each item for sale;
  • a data request is sent to Amazon via Yahoo! pipes for more information on each book;
  • further Yowl notifications for each eBay item and book are shown, using the title and thumbnail retrieved.
Note that there is a bug in YUI which will cause the eBay tooltip to be nothing but a flashing white box if the browser window is too small.
