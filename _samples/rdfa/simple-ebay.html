eBay

This file contains a reference to an item for sale on eBay, identified with RDFa. The item number provides the 'hook' for the UI processor to add:

  • an embedded widget that gives details of current bids for the item;
  • a Yowl notification that an eBay item has been found on the page;
  • a data request to the eBay API for more information on the item;
  • a further Yowl notification that uses the title and thumbnail retrieved dynamically.
